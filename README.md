# AstrBot Group Monitor Plugin

监控群成员退群事件，并发送通知到指定的管理群。

## 功能
- **监控群列表**：查看当前监控的群组及对应的通知群。
- **添加/删除监控**：灵活配置哪些群需要监控，以及通知发送给谁。
- **自动关联**：添加监控群时，若通知群未在列表中，会自动添加。
- **详细通知**：包含退群时间、群号、QQ号、昵称及退群类型（主动退群/被踢）。

## 指令
- `/群变动菜单`：查看帮助
- `/通知群列表`
- `/添加通知群 <群号>`
- `/删除通知群 <群号>`
- `/监控群列表`
- `/添加监控群 <监控群号> <通知群号>`
- `/删除监控群 <监控群号>`
